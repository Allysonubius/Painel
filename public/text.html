Painel de informações sobre o Covid-19

       O objetivo desta página e destacar o desenvolvimento da Covid-19. em uma aplicação resumida fazendo a emissão de relatórios sobre o Covid-19.

    Esta aplicação permite que o usúario faças as seguintes escolhas:

      Casos confirmados acumulados;
      Novos casos no dia;
      Obitos acumulados;
      Obitos no dia;
      Indice de Mortalidade(%)
      Indice de recuperação(%)
    O Front-End desta aplicação foi desenvolvido em VueJs, desenvolvido de um projeto no GitHub Open Source COVID 19 Original Open Source.
    O codigo esta disponibilizado no Github acesse em Covid 19 Timeline - PT-BR

Explicação sobre o projeto por Allyson - Desenvolvedor Full Stack iniciante.
